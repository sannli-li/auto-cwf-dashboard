# Auto CWF Dashboard

🚀 **Live Dashboard**: [View Dashboard](https://你的用户名.github.io/auto-cwf-dashboard/)

## 📊 Features

- **Interactive Charts**: Visualize CWF data with dynamic charts
- **Real-time Filtering**: Filter by date, department, and status
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Export Data**: Download filtered data as CSV

## 🛠️ Technology Stack

- Pure HTML/CSS/JavaScript
- Chart.js for visualizations
- No backend required - fully static

## 📝 Data Source

Data extracted from Qualcomm Orbit Query 112709

## 🔄 Updates

To update the dashboard:
1. Replace `index.html` with new version
2. Commit and push changes
3. GitHub Pages will auto-deploy in 1-2 minutes
